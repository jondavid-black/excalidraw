# Quickstart: Upgrade NextJS to 16.0.1

## Steps

1. Checkout branch `002-upgrade-nextjs`.
2. Update NextJS dependency to version 16.0.1 in all relevant `package.json` files.
3. Run `npm install` or `yarn install` to update lockfiles.
4. Run all automated tests (`yarn test:app` or equivalent).
5. Verify application builds and runs successfully.
6. Update documentation references to NextJS 16.0.1.
7. Commit and push changes.

## Notes

- Ensure all dependencies are compatible with NextJS 16.0.1.
- Review changelogs for breaking changes and required code updates.
- Validate container builds and health endpoints post-upgrade.
